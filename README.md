# Event-_Management_System-
This is a smart event organizer digitally and in one place.
EventPulse  | Smart Event Management System
EventPulse Pro is a high-fidelity, data-driven management platform designed for college organizers to plan, track, and execute campus events with precision. It replaces fragmented communication and manual spreadsheets with a unified, professional dashboard.

Problem Statement Relevancy: Traditional event planning suffers from budget leaks and task delays. EventPulse centralizes these workflows into a single-page application (SPA) for real-time oversight.


Key Modules:
Organizer Auth Gate: Secure login portal restricted to authorized personnel.

Insightful Dashboard: Interactive data visualization using Chart.js for registration trends and attendance.

Dynamic Budgeting: Categorized expense tracking with real-time balance calculation and visual analytics.

Task Board: Progress-driven task management with priority badges and circular completion indicators.

Event Timeline: Chronological schedule manager with a visual nodes-and-path timeline.

Tech Stack
Frontend: HTML5 (Semantic), CSS3 (Grid & Flexbox), JavaScript (ES6+).

Visualization: Chart.js for professional business intelligence graphs.

Design Pattern: Single Page Application (SPA) architecture for fluid transitions.
